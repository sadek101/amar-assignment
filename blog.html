Discuss the scope of var, let, and const

var: var is a keyword used to declare a variable. It stands for variable and it's used to hold different types of values such as numbers, strings, objects, etc.

let: let declarations are scoped to blocks as well as functions.

const: const declarations cannot be redeclared by any other declaration in the same scope.

Tell us the use cases of null and undefined

undefined: undefined is the default value of a variable that has not been assigned a specific value, or a function that does not return any value

null: null is an assignment value that represents no value or no object

What do you mean by REST API?

Each resource in a REST API is identified by its URL, and the API uses these URLs to interact with the resources.
